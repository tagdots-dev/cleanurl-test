# ⭐ _origin_url_

### ✅ Purpose: get origin URL address.

<br>

```
Return the combination of protocol, fqdn, and port without redirection.
```

<br>

### 💥 Running in Python interactive runtime environment

### Import client library
```
>>> from pkg_19544 import origin_url
```

### Run origin_url
```
>>> user_url = 'https://google.com/search?q=hello+world'

>>> origin_url(user_url)
'https://www.google.com'
```
